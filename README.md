
# Libre-Case

  
### To set venv() to the .venv directory 

    python -m venv .venv

  
### To activate venv 

    .venv\Scripts\activate.bat

  

### To deactivate venv

    .venv\Scripts\deactivate.bat

  
### To backup libraries

    pip freeze > requirements.txt

  
### To install libraries

    pip install -r requirements.txt
